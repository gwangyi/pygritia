pygritia.call.lazy_call(func: _T) → _T[source]¶

Make a function to support lazy expression

>>> from pygritia import this, lazy_call, evaluate
>>> @lazy_call
... def hello(obj):
...     print(obj)
>>> print(hello(this))
hello(this)
>>> hello("123")
123
>>> evaluate(hello(this), {this: "123"})
123
